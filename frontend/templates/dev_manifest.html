dealers.json

Fetches the dealer homepage when possible; otherwise paste HTML so the same heuristics can run without a server-side fetch. Review the new row before saving.

Name URL Provider Dealer ID Brand (optional)

Provider dealer_dot_com and dealer_on match the Python parsers under backend/parsers/. Dealer ID is a lowercase slug (used in API paths and stored on vehicle rows). Look-up uses heuristics on the downloaded HTML; wrong platform or ID is possible on unusual sites. Use Scan on a row to run python scanner.py --dealer-id … for that dealer (save dealers.json first; watch the Flask terminal for logs).

Quick reference

  • Web app: python run.py — default http://127.0.0.1:5000
  • Scanner: python scanner.py — reads dealers.json, writes inventory DB
  • Enable this page: set environment DEV_CONSOLE=1. Optional: DEV_CONSOLE_SECRET plus sign-in at /dev/manifest/login
