Developer console is off

The dealers manifest console (/dev/manifest) is disabled until you enable it for this process.

In the same terminal where you run the app, run:

export DEV_CONSOLE=1
then start again: python3 run.py

Optional lock: export DEV_CONSOLE_SECRET='your-secret' — then open /dev/manifest/login.

If the page still fails after that, install dependencies: pip install beautifulsoup4 requests (needed for “Look up” from URL).
