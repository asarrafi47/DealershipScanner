Smart import

Bulk magic URLs

Runs node scanner.js --url … --smart-import (Dealer.com search API + getInventory, lazy-load scroll). When inventory is persisted, adds or updates project-root dealers.json so python scanner.py --dealer-id … picks up the dealer. The registry row (geocode + listings filters) is written only when discovery returns a valid city/state; until then, listings may omit incomplete vehicles.
